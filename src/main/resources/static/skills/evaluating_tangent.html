Evaluating tangent

We're going to get fluent with the values of the tangent function at multiples of `pi/6` and `pi/4`.

Mental representations

While practising this skill, deliberately try and hold these mental representations in your head as you think about each question.

If you get a question wrong, re-visit and update your mental representation.

Recalling sine and cosine

The tangent function is defined to be (sin A)/(cos A) for some angle A. So you need to fluently evaluate the sine and cosine function.

If you're struggling, do some deliberate practice of those skills.

Imagining the gradient of the line

The tangent function of an angle is also the gradient of the line created by that angle with the horizontal.

So picture the shape of the line created by the angle and ask yourself what its gradient must be.

For example, we can tell instantly that the line y = x has a gradient of 1, so tan pi/4 = 1.
